4Bytes Framework

4Bytes is a super easy, light-weight PHP framework. It isn't really object-oriented, it doesn't have to be.

If you have a small project that requires PHP, 4bytes could be the ketchup to your hotdog.

About 4Bytes Framework

4Bytes is designed to be easy, and fast to develop with.
It comes with a JSON file database for easy getting/storing.

The main goal of 4Bytes is ease-of-use. Nothing is more frustrating, especially to a new PHP developer, than trying to learn a complex framework on top of learning PHP. With the 4Bytes Framework, you get a really powerful system, whilst being easy to read/code.

Check out the code snippets below to get a taste of 4Bytes:

Here is your basic 'Hello, World!' program.
Edit your /views/index.php file.

  1. Hello, World!

Holy sugar pickles, that was easy! No routing, no weird namespaces or request controllers. 4bytes is about being easy to learn, and master.

Let's checkout the built-in JSON database api.
Open the index controller in controllers/index.php.

  1. $db->set('settings','SITE.NAME', '4bytes');
  2. $this->vars['site_name'] = $db->get('settings', 'SITE.NAME');

Now go into your view:

  1. <?php print $site_name; ?>
  2. // output: 4bytes
You can check the stored values in the database folder.
