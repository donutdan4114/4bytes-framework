About 4bytes

4bytes is designed to change the way you make PHP applications. This easy-to-use framework is designed around being the fastest, easiest, and prettiest PHP framework in all the land.

Perhaps you didn't know that a PHP framework can be pretty, well here's proof:


//create a menu item for this mymodule.inc:
function mymodule_menu(){
  return array('blog'=>'Blog');
}



//in the mymodule.blog.controller.inc file:
class Blog{

  function __construct(){

    print new Template(
      'mymodule.blog', //name of the template
      $args = array(), //array of args to pass
      $cache_time = 300, //cache time in seconds
    );

  }
	
}



//in your mymodule.blog.template.php file:
<p>hello, world!<p>
